# LifeAligned AI v1

LifeAligned AI v1 is a web-based cognitive alignment assessment tool that helps users understand how they think, where they perform best, where they are misaligned, and what realistic steps they should take next.

## Features
- 12-question assessment
- Structured cognitive profile
- Capability extraction
- Direct insight
- 7-day action plan

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- OpenAI API

## v1 Goal
Validate that users feel accurately understood and receive useful next steps.
